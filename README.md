# Alura_consumindo_API_omdb
Esse projeto em console, desenvolvido com base na recomendação do curso "Spring Boot 3: desenvolva uma API Rest em Java", busca praticar tais conteúdos, mediante o consumo de uma API de filmes e séries OMDb, com base no link HTTP passado. Em seguida, gera-se um formato JSON com os dados retornados.
